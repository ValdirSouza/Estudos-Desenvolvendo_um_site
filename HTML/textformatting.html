Falando Sobre Tags de Formatação de Texto

- Introdução

As Tags HTML são instruções para estruturar e criar páginas web. Elas fornecem para o navegador utilizado uma maneira de como definir a estrutura e o layout de uma página, incluindo vários elementos como por exemplo texto, imagens, links, entre outros.
As Tags HTML são essenciais para criar qualquer documento HTML e sem ela, os documentos HTML não seriam exibidos de maneira correta e não teria contéudo algum.

- Formatações

- Bloco de Citação

- Fonte

A tag <fonte> por si só, não formata um texto ou trecho que desejar em HTML. Essa tag precisa de atributos para poder está realizando as configurações necessárias.
Logo abaixo terá algumas citações de atributos que podem ser utilizados e como iram comportar, porém esse tipo de estilização pode ser realizada utilizando códigos CSS e é mais viável.

- Itálico

O método para formatar um texto em HTML em itálico é a Tag "< i>". Normalmente o itálico é utilizado em citações, algumas palavras como termos regionais, para diferenciar o texto.

- Marcação

O método para formatar um texto em HTML com marcação de texto é a tag <mark>.

- Negrito

O método para formatar um texto em HTML em negrito é a Tag "< b>" ou a Tag "< strong>". Quanto a visualização, não tem diferença entre eles, iram gerar o mesmo resultado no navegador web, já no código, o strong é utilizado para dar ênfase naquele trecho do texto.

- Sobrescrito

O método para formatar um texto em HTML com sobrescrito é a Tag "< sup>". Está Tag é para que uma string seja exibida como sobrescrito, ou seja, escrever um texto com letras menores e com o alinhamento um pouco acima das que está no conteúdo do texto, como por exemplo:

  • área2
  • volume3
  • textoteste

- Subescrito

O método para formatar um texto em HTML com subescrito é a Tag "< sub>". Está Tag é para que uma string seja exibida como subrescrito, ou seja, escrever um texto com letras menores e com o alinhamento um pouco abaixo das que está no conteúdo do texto, como por exemplo:

  • log10
  • 102
  • textoteste

- Sublinhado

O método para formatar um texto em HTML com sublinhado é a Tag "< u>".
