Falando Sobre Links

- Introdução

Links no HTML são hiperlinks, ou seja, são links representados por uma palavra, texto ou imagem que quando o usuário clica, o encaminha para outras páginas web, arquivos, endereços de e-mails, ligações na mesma página ou endereços na URL, podendo levar por exemplo a conversas em aplicativos.

- Atributos

download

Este atributo direciona o usuário a salvar o documento que está referenciado no link em vez de nagevar por ele e pode ser usado com valor ou sem.

  • Definindo um valor para esse atributo, ele irá sugerir como nome do arquivo. Caso tenha caracteres especiais que algum sistema não aceite em nome de arquivos, então o navegador irá ajustar o nome para o sugerido caso necessário.
  • Não definindo um valor o navegador irá solicitar ao usuário um nome e/ou extensão para que o arquivo seja salvo
href

Este atributo sendo o mais importante do elemento é necessário para efetuar a conexão especificando uma localização para a qual o usário irá ir no navegador.
A princípio o link ficará visível para o leitor, caso o mesmo não utilize um título para sobrepor o link, mas o link não deixara de aparecer, basta o usuário colocar o mouse sobre o link que o mesmo irá aparecer no rodapé do navegador que estiver sendo utilizado.

target

Este atributo especifica onde irá abrir o documento vinculado ao clicar no link.
Tem se 4 valores que possam ser dados a este atributo, começando do primeiro que é o mais usado até o último que é o menos usado:

  1. _self: Abre o documento na mesma guia/janela em que foi clicado. Esse tipo de comportamento é o padrão da tag "a"
  2. _blank: Abre o documento em uma nova guia/janela
  3. _top: Abre o documento do contexto de navegação mais alto que antecesde o atual. Se não tiver antepassados, comporta-se como o _self
  4. _parent: Abre o documento do contexto de navegação pai do atual. Se não tiver nenhum pai, comporta-se como o _self
title

O atributo title ele irá trazer um "balão" de informações sobre o link a colocar o mouse sobre o texto a ser acessado, essas informações são as que colocamos no texto do atributo title. Esse "balão" é mais conhecido como tooltips.
