Introdução

A sigla HTML significa HyperText Markup Language/Linguagem de Marcação de Hipertexto, que é uma linguagem de marcação utilizada em páginas da Web para poder criar e estruturar seções, parágrafos, links, tags e atributos, mas ele não é considerado uma linguagem de programação por não poder criar funcionalidades dinâmicas como o Python, por exemplo. *
