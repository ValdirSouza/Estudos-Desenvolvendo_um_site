Falando Sobre Tipos de Listas

- Listas de descrição

A Tag "dl" é para definer uma estrutura de listas de descrição.
A Tag "dt" pode ocorrer somente sendo filha da tag anterior que foi citada e irá definer os termos da lista.
Para descrever cada termo ira se usar a Tag "dd".
Como por exemplo:

1 lista de descrição
primeira descrição
segunda descrição
2 lista de descrição
primeira descrição
3 lista de descrição
primeira descrição
segunda descrição
terceira descrição

- Listas não ordenada

A Tag "ul" é para definer uma estrutura de listas não ordenada
A Tag "li" irá definer os itens da lista.
Esse tipo de lista pode ter uma estrutura de listas dentro de outras.
Os itens terão marcadores do tipo simbolos pré-definidos, mas podendo ser removidos.
Como por exemplo.:

  • 1 lista
  • 2 lista
  • 3 lista
    • 3.1 lista
    • 3.2 lista
      • 3.2.1 lista
        • 3.2.1.1 lista
        • 3.2.1.2 lista
          • 3.2.1.2.1 lista
      • 3.2.2 lista
    • 3.3 lista
  • 4 lista

- Listas ordenadas

A Tag "ol" é para definer uma estrutura de listas ordenada.
A Tag "li" irá definer os itens da lista assim como na lista não ordenada.
Nesse tipo de lista é possível montar uma estrutura de listas dentro de outras.
Por padrão os itens terão marcadores com números, mas podendo ser letras ou números romanos.
Além de poder retirar os marcadores como na lista anterior, é possível alterar os marcadores. Como por exemplo.:

  1. - 1 lista
  2. - 2 lista
  3. - 3 lista
    1. - 3.1 lista
    2. - 3.2 lista
      1. - 3.2.1 lista
      2. - 3.2.2 lista
        1. - 3.2.1.1 lista
        2. - 3.2.1.2 lista
          1. - 3.2.1.2.1 lista
    3. - 3.3 lista
  4. - 4 lista
    1. - 4.1 lista
    2. - 5.1 lista
      1. - 5.1.1 lista
      2. - 5.1.2 lista
  5. - 5 lista
