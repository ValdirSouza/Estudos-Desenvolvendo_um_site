Falando Sobre Tipos de Listas

- Listas de descrição

A Tag <dl> é para definer uma estrutura de listas de descrição.
Já a Tag <dt> pode ocorrer somente sendo filha da tag anterior que foi citada e irá definer os termos da lista.
Para descrever cada termo se utiliza a Tag <dd>.
Como por exemplo:

1 lista de descrição
primeira descrição
segunda descrição
2 lista de descrição
primeira descrição
3 lista de descrição
primeira descrição
segunda descrição
terceira descrição

- Listas não ordenadas

A Tag <ul> é para definer uma estrutura de listas não ordenada e a <li> irá definer os itens da lista.
Esse tipo de lista permite ter uma estrutura de listas dentro de outras. É definida como lista não ordenada, pois os itens terão marcadores do tipo simbolos que sáo pré-definidos, mas os marcadores podem ser removidos ou trocados mudando o valor do atributo style dando o valor para ele de "list-style-type:" e acrescentando "none", "circle", "georgian", erc. Isso pode ser feito para toda lista ou apenas um item da lista ao qual desejar, mas atenção, quando é aplicado para a lista toda, as lista subsequentes não sofrem essa aplicação.
Como por exemplo.:

  • 1 lista
  • 2 lista
  • 3 lista
    • 3.1 lista
    • 3.2 lista
        list-style-type:none
      • 3.2.1 lista
        • 3.2.1.1 lista
        • 3.2.1.2 lista
          • 3.2.1.2.1 lista
        • 3.2.1.3 lista _ list-style-type:circle
      • 3.2.2 lista
    • 3.3 lista
  • 4 lista

- Listas ordenadas

A Tag <ol> é para definer uma estrutura de listas ordenada e a <li> irá definer os itens da lista assim como na lista não ordenada.
Assim como na lista não ordenadas, essa também é possível montar uma estrutura de listas dentro de outras. Por padrão os itens terão marcadores numéricos, além de poder retirar os marcadores como na lista anterior, é possível alterar os marcadores, podendo ser letras, números romanos ou iniciar de um valor númerico qualquer e seguirar a ordem.
Para mudar o valor númerico específico que deseja que o marcador ordenado inicie, deve-se declarar o atributo start e alterar o valor, atribuindo o número que desejar. Já para alterar o tipo, deve-se declarar o atributo type e alterar o valor para i maiúsculo ou minúsculo para o marcador se tornar um número romano ou a maiúsculo ou minúsculo para o marcador se tornar letras em ordem alfabética. Para ambas configurações, o tamanho irá variar entre maiúscula ou minúscula respectivamente de acordo como declarar o valor no atributo. Como por exemplo.:

  1. - 1 lista
  2. - 2 lista
  3. - 3 lista
      start="4"
    1. - 3.1 lista
    2. - 3.2 lista
        type="i"
      1. - 3.2.1 lista
      2. - 3.2.2 lista
          type="a"
        1. - 3.2.1.1 lista
        2. - 3.2.1.2 lista
            list-style-type:none
          1. - 3.2.1.2.1 lista
    3. - 3.3 lista
  4. - 4 lista
      type="I"
    1. - 4.1 lista
    2. - 5.1 lista
        type="A"
      1. - 5.1.1 lista
      2. - 5.1.2 lista
  5. - 5 lista
