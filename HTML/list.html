Falando Sobre Tipos de Listas

- Listas de descrição

A Tag "dl" é para definer uma estrutura de listas de descrição.
A Tag "dt" pode ocorrer somente sendo filha da tag anterior que foi citada e irá definer os termos da lista.
Para descrever cada termo ira se usar a Tag "dd".
Como por exemplo:

1 lista de descrição
primeira descrição
segunda descrição
2 lista de descrição
primeira descrição
3 lista de descrição
primeira descrição
segunda descrição
terceira descrição

- Listas não ordenada

A Tag "ul" é para definer uma estrutura de listas não ordenada
A Tag "li" irá definer os itens da lista.
Os itens desse tipo de lista terão marcadores do tipo simbolos ♦.
Com esse tipo de estrutura é possível montar de listas de descriçã dentro de outra lista.
Como por exemplo.:

  • 1 lista
  • 2 lista
  • 3 lista
    • 3.1 lista
    • 3.2 lista
      • 3.2.1 lista
      • 3.2.2 lista
    • 3.3 lista
teste
  1. 1 lista
  2. 2 lista
  3. 3 chamar segundo nivel
    • 3.1 lista
    • 3.2 chamar terceiro nível
      • Este é um terceiro nível
      • 3.3.
    • 3.3 lista
