Falando Sobre CSS

- Introdução

CSS significa Cascading Style Sheets (Folha de Estilo em Cascatas). Ele tem esse nome porque aplica-se as estruturas de “cima para baixo”, ou seja, por meio de regras com prioridades para seguir. O recurso é utilizado para estilizar documentos e facilmente aplicado com as linguagens de marcação HTML, XML e XHTML.

- Formas de declarar o CSS

No processo de declaração de estilos em CSS, uma hierarquia é seguida, na qual cada regra tem sua própria prioridade sobre as outras. Por exemplo, a estilização aplicada diretamente no atributo de uma linha tem maior prioridade do que a declarada internamente no documento, e esta, por sua vez, tem mais prioridade do que a estilização definida em um documento externo.
Em resumo segue as explicações sobre cada tópico:

  • Inline
    • Interno
      • Externo

- Inline

Adiciona o código CSS utilizando o atributo <style> dentro das tags HTML, elemento por elemento para poder personalizar e montar a configurações que desejar.

- Interno

O código é adicionado dentro da tag <head> da página do HTML utilizando a tag <style> e dentro dessa tag é atribuido as regas de estilização do CSS.

- Externo

É criado um arquivo com a extensão .css com todas as regras de estilização que queira aplicar e posteriormente esse arquivo CSS é referenciando no HTML da página através da tag <link>.

- Tipos de Seletores

Básicamente um Seletor ele vai definir em qual elemento vai aplicar as propriedades CSS.

- Seletor de Tags

O Seletor de Tags ou também conhecido como, Seletor de Tipo busca os elementos referenciando pelos nomes das tag's padrão do HTML que irá estilizar.

- Seletor por ID

O Seletor por ID busca elementos referenciando através do atributo "id". Para isso deve se usar o valor que foi atribuido ao id. Ele deve ser utilizado com se fosse uma impressão digital de um elemento, por isso devemos usar o atribuido id apenas uma unica vez por página, não se repetindo.

- Seletor por Classe

O Seletor por Classe busca elementos referenciando através do atributo "class", ou seja, o valor dado a class. Ele é uma forma utilizada quando você quer identifcar um grupo de elementos, podendo-se repetir dentro de um mesmo documento. Através das classes é possível atribuir estilos em vários elementos do HTML de uma vez só.

- Seletor Universal

O Seletor Universal busca todos os elementos HTML.

- Seletor de Atributo

O Seletor de Atributo busca elementos que possuem um atributo específico no documento HTML e consegue também buscar atributos com um valor específico no documento.
