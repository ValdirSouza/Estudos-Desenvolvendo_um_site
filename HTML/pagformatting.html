Falando Sobre Formatação de Página

- Introdução

As Tags de formatação de página são elementos de bloco utilizadas para agrupar, organizar e aplicar estilos em outros elementos específicos do contéudo do documento de uma página web. Sendo que as Tags tem propósitos e comportamentos diferentes.
Elas são essenciais para a estruturação e o layout de uma página, permitindo criar seções distintas e aplicar estilos ou scripts.
