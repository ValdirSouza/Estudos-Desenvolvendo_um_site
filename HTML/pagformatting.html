Falando Sobre Formatação de Página

- Introdução

As Tags de formatação de página são elementos de bloco utilizadas para agrupar,organizar e aplicar estilos em outros elementos específicos do contéudo do documento de uma página web. Sendo que as Tags tem propósitos e comportamentos ligeiramente diferentes. Elas são essenciais para a estruturação e o layout de uma página, permitindo criar seções distintas e aplicar estilos ou scripts.
Essas tags são distintas e usadas para estruturar as páginas web, sendo que não tem peso semântico, apenas delimitando o espaço e onde irá aplicar as configurações.

- Elemento de Divisão

- Tag HTML <div>

A Tag <div> por padrão é um elemento de bloco, que simplismente ocupa toda a largura disponível e possui quebra de linhas antes e depois.

- Tag HTML <span>

A Tag <span> por padrão é um elemento de linha, que apenas ocupa o trecho ao qual desejar e não tem quebra de texto no início ou final.

- Tag HTML <fieldset>

A Tag <fieldset> é utilizada para agrupar elementos relacionados em um formulário ou elementos que desejar. Está tag cria uma caixa ao redor dos elementos relacionados.

- Tag HTML <embed>

A Tag <embed> é um tipo de tag HTML para medias externas, usada para incorporar arquivos multimedia de áudio e vídeo. Uma música ou vídeo podem tocar quando uma página é exibida. Portanto, pode-se inserir arquivos de som e vídeo dentro de um documento HTML, há extensões cujos arquivos são grandes, exigindo um tempo maior de download.
Porém essa tag por questão de semântica e em caso de um robô acessar o site, ele não irá reconhecer que a tag embed se trata de um vídeo, devido a essa situação o ideal é utilizar a Tag <video> e dentro de sua estrutura a Tag <source> com o direcionamento do local do arquivo a ser reproduzido.

- Tag HTML <iframe>

A Tag <iframe> é um tipo de tag em um documento HTML que vibializa a inclusão de outro documento HTML dentro do primeiro, ou seja, na mesma página. Basicamente ele permite que a partir de uma página inicial, seja possível abrir uma seção de outra página qualquer. Ou seja, ele carrega médias externas, como do youtube e pode carregar até sites inteiros.
Essa tag tem suas particularidades de segurança, por isso é importante ter cuidado com a segurança do site ao utilizar essa tag e fazer tratativas via JavaScript.
